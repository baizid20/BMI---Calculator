# BMI Calculator

A simple, responsive, and user-friendly **BMI (Body Mass Index) Calculator** built with **HTML, CSS, and JavaScript**. This application helps users quickly calculate their BMI based on their height and weight and provides an easy-to-understand health status.

## ✨ Features

* Calculate BMI instantly
* Simple and clean user interface
* Responsive design for desktop and mobile
* Real-time input validation
* Displays BMI value along with weight category:

  * Underweight
  * Normal Weight
  * Overweight
  * Obese
* Fast and lightweight application

## 🚀 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## 📖 How BMI is Calculated

BMI is calculated using the following formula:

**BMI = Weight (kg) / Height² (m²)**

Example:

* Weight: 70 kg
* Height: 1.75 m / 5 foot
* BMI = 70 / (1.75 × 1.75) = **22.86**

## 🎯 Purpose

The goal of this project is to provide a simple and accessible tool that allows anyone to calculate their Body Mass Index quickly and understand their general weight category.

> **Note:** This calculator is intended for educational and informational purposes only. It should not replace professional medical advice.
